# 訂便當系統
泰山職訓訂便當系統 ([線上範例](https://dinbendon.kento520.tw/))   

![預覽圖](https://github.com/rogeraabbccdd/PHP-Dinbendon/raw/master/preview/dinbendon.jpg)

## 功能
* 統計便當
* 新增便當店及修改菜單
* 便當店評價
* 依照班級分流
* 可愛小狗狗

## 安裝
* PHP版本需求為7.0以上
* 將檔案放入網頁跟目錄後，開啟`assets/inc/config.php`修改 MySQL 的帳號密碼
* 預設使用者帳號密碼
  * 1234、1234
  * 5678、5678
